## Set up a page wrapper

### The page structure
```
body.l-pageParallax 

  .navBar = Page Navigation

  .pageContentWrapper = content

    banner = Site banner

    pageContent = Page content

     contentSection 
     contentSectionImage
     
     contentSection 
     contentSectionImage
     
     contentSection 
     contentSectionImage
     
     contentSection 
     contentSectionImage

    pageFooter = Page footer
```
